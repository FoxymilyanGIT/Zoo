## Seed data overview

Эта папка содержит человеко-читаемые JSON-файлы с начальными данными, которые синхронизированы с миграцией Flyway `sql/migrations/V1__init_schema_and_seed.sql`.

- `animals.json` — 10 животных каталога
- `events.json` — 3 ближайших события/мероприятия
- `users.json` — учетные записи admin/user (пароли хранятся в явном виде для удобства, в базе данных сохраняются в BCrypt)

Flyway автоматически применит миграции и заполнит БД при выполнении `mvn spring-boot:run`. Если требуется повторно заполнить БД, очистите таблицы и выполните миграции заново или импортируйте данные из этих файлов любым удобным способом.
